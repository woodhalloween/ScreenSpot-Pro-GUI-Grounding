## Set Up

Before you begin, ensure your environment variables are set:

- `OPENAI_API_KEY`: Your OpenAI API key.

## Evaluation
Use the shell scripts to launch the evaluation. 
```bash 
bash run_ss_pro.sh
```
or
```bash 
bash run_ss_pro_cn.sh
```
